# Interns evaluation
## Features
Intern part : 
 * Answer quizzes
 * See stats about himself

Administrator part :
 * Add & manage user
 * Manage subject, quizzes, questions & answer : creation, deletion, modification
 * See interns score for each answered quizzes for each of them
 * Also answer quizzes
